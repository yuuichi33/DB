# 阶段1：需求冻结与验收清单

本清单用于对应课程要求，开发与截图都按此执行。

## 1. 页面要求

- [ ] 首页（导航）
- [ ] 商品列表页面
- [ ] 用户列表页面
- [ ] 订单列表页面
- [ ] 查询展示页面（基本查询、连接查询、聚合分组、视图）

## 2. 数据库定义

- [ ] 创建数据库
- [ ] 创建三张表：user、item、orders
- [ ] 设置主键
- [ ] 设置非空、外键、唯一、状态合法性约束

## 3. 一致性规则

- [ ] 每个商品最多交易一次
- [ ] orders.item_id 只能出现一次

- [ ] status=0 的商品不能出现在 orders

## 4. 数据操作

- [ ] 插入给定初始数据
- [ ] 插入一个新商品
- [ ] 修改某商品价格

- [ ] 操作后页面刷新可看到变化
